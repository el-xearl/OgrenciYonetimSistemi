# Öğrenci Yönetim Sistemi

Bu proje, temel bir **Öğrenci Yönetim Sistemi** uygulamasıdır.  
Öğrencilerin bilgilerini ekleme, silme, listeleme ve güncelleme işlemleri yapılabilir.

---

## Hakkında

- Windows üzerinde **C# ve .NET** kullanılarak geliştirilmiştir.  
- CRUD (Create, Read, Update, Delete) işlemlerini destekler.  
- Eğitim ve örnek proje amaçlı hazırlanmıştır.

## Kurulum ve Çalıştırma

1. Projeyi GitHub’dan klonlayın:

```bash
git clone https://github.com/kullaniciadi/OgrenciYonetimSistemi.git
````

2. Visual Studio veya Visual Studio Code ile projeyi açın.
3. Projeyi çalıştırın:

   * **Visual Studio:** `F5` tuşuna basarak çalıştırabilirsiniz.
   * **Visual Studio Code:** Terminalde `dotnet run` komutunu kullanabilirsiniz.

---

## Kullanım

* Program açıldığında menüden öğrencileri **ekleyebilir, silebilir veya listeleyebilirsiniz**.
* Öğrenci eklerken **isim, soyisim ve öğrenci numarası** gibi bilgiler girilmelidir.
